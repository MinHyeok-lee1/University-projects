# flutter-opencv

Integration experiment combining a Flutter frontend with Python OpenCV backend.

## Features

- Flutter for UI
- Python backend for OpenCV logic
- Communication via HTTP (or future WebSocket)

## Folder Structure

- `lib/`: Flutter source
- `app.py`: Flask server with OpenCV logic

## Tech Stack

- Flutter (Dart)
- Python
- Flask
- OpenCV
